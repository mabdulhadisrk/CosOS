# LarOS
initially names it as CosOS, but to make it fun and interesting I made the theme etc to represent my fav character from tekken Lars. It was the hardest project for me uptil now.
# Languages
Html: created the skeleton for the code, and a div for preface
css: beautiful adding of blurs to top and bottom bars, beautiful transitions and styling.
js: hardest part for the logic ( especially the weather part )
# Reason of making
i was bored to make boring projects who are made by many people, so I wnated to create new and interesting thing representing my taste, so I made WebOS which stardance posted as a challenge.
# Features
beatiful transitions. taskbar to run my favourite daily use apps. current battery, secured network and the hardest part making the current temperature teller. current time teller least count 1 second.
